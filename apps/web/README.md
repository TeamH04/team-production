# Web (Next.js)

Next.js 15 (App Router) の店舗探索アプリ。

## クイックスタート

```bash
# リポジトリルートから
make web    # Web 起動
```

## コマンド

| コマンド                    | 説明             |
| --------------------------- | ---------------- |
| `make web`                  | 開発サーバー起動 |
| `make test-web`             | テスト実行       |
| `pnpm --dir apps/web build` | ビルド           |

## 画面構成

### トップページ `/`

- 検索ボックスとカテゴリチップで絞り込み
- 「さらに読み込む」で 9 件ずつ追加表示

### 店舗詳細 `/shop/[id]`

- 画像スライダー、タグ、メニュー、地図リンク
- 共有/お気に入りボタン

## ポート
