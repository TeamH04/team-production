# Web フロントエンド

Next.js で構築した Web アプリです。店舗データはワークスペースパッケージ `@team/shop-core` から読み込みます。

## 開発の始め方

```bash
pnpm install
pnpm --dir apps/web dev
```

ローカルでは http://localhost:3000 を開くとトップページ、 http://localhost:3000/shop/[id] で店舗詳細ページを表示できます。`id` には `@team/shop-core` の `SHOPS` に含まれる店舗 ID（例: `shop-1`, `shop-2` ...）を指定してください。

## 主な機能

- トップページ `/` で店舗一覧を表示。検索キーワードとカテゴリチップで絞り込み、タグをクリックするとキーワードに反映されます。カードをクリックすると店舗詳細へ遷移します。
- 店舗詳細ページ `/shop/[id]` を実装済み。店舗名、画像スライダー、タグ、メニュー、お気に入り/共有ボタン、レビュー案内を表示します。
- 画像は Unsplash などのリモート画像を利用するため、`next.config.ts` にリモートドメインを許可済みです。
- 存在しない店舗 ID ではカスタム 404 ページを表示し、一覧への導線を提示します。

## 品質チェック

```bash
pnpm --dir apps/web lint
```

Lint が通ることを確認してからプルリクエストを作成してください。
